# DreamIT Biz — 전체 사이트 총 정리

> **Ph.D 이애본 (Aebon Lee)** | Claude와 함께 개발 (2025.02.19~)
> 마지막 업데이트: 2026-05-01 | 총 **87개** 사이트

---

## 프로젝트 소개

**DreamIT Biz**는 교육, 경영, IT, 인공지능 등 다양한 분야를 아우르는 종합 교육 플랫폼입니다.
2025년 2월 19일부터 Claude AI와 함께 개발을 시작하여, 현재 **87개의 전문 학습 사이트**를 운영하고 있습니다.

### 기술 스택
- **프론트엔드**: React 19 + Vite 7~8 + TypeScript 5.8
- **백엔드**: Supabase (PostgreSQL + Auth + Edge Functions)
- **결제**: PortOne Browser SDK (KG이니시스) — 24개 사이트
- **배포**: GitHub Pages (`*.dreamitbiz.com`)
- **테마**: 다크/라이트/자동 + 5색 컬러 테마 + 한/영 전환
